AI Chat Bot with LangChain + Gradio

Goal: Use LangChain & OpenAI LLM module to call OpenAI and display response.

Tools: LangChain + Gradio + OpenAI API

A Gradio chat UI that accepts user prompt and returns LLM response.

Features
 -  Interactive chatbot UI using Gradio

 -  LLM integration via OpenAI API

- Modular pipeline using LangChain (LLMChain)

- Prompt templating for structured input

- Fast and lightweight deployment (local or cloud-ready)
